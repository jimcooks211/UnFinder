# UnFinder

Built with Vite + React.

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
UnFinder/
├── public/
│   └── unfinder.svg        # Favicon
├── src/
│   ├── assets/             # Static assets
│   ├── App.jsx             # Root component
│   ├── App.css             # App styles
│   ├── main.jsx            # React entry point
│   └── index.css           # Global styles
├── index.html
├── vite.config.js
└── package.json
```

## Stack

- **Vite** v5 — lightning fast dev server & bundler
- **React** v18 — UI library
- **ESLint** — code linting

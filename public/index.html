iOS Installation Guide

Get UnFinder
on your iPhone

Three ways to install — no App Store needed. Pick whichever feels easiest.

⭐ Recommended
Install via Profile
Downloads a Web Clip that places the UnFinder icon directly on your home screen.
✓ Works right now
1
Tap the download button below
Downloads a small .mobileconfig file — only contains a shortcut to UnFinder.
2
Allow the download
Safari asks "Allow website to download configuration profile?" — tap Allow.
3
Open Settings → Profile Downloaded
A banner appears at the top of ⚙️ Settings. Tap it, then tap Install top-right.
4
Done — UnFinder is on your home screen
Tap the icon anytime to open the app instantly.
Download Install Profile
🔒 This profile only adds a home screen bookmark. It does not collect data or modify device settings.
✓ Works right now
Add via Safari
Open UnFinder in Safari and use the Share menu to add to your home screen.
1
Open in Safari specifically
Must be Safari — other browsers won't show the option.
2
Tap the Share button ⬆️
Bottom toolbar on iPhone, top-right on iPad.
3
Tap ＋ Add to Home Screen
Scroll through the share sheet to find it.
4
Confirm name and tap Add
Open UnFinder in Safari
Alternative Methods
Not using Safari? Here are other ways to install UnFinder.
🔵 Chrome for iOS
1
Open UnFinder in Chrome
2
Tap ⋯ More bottom-right
3

1

2

4
Tap ⋯ Details → Add to Home Screen
or

1
Open UnFinder in Edge
2
